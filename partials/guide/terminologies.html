A terminology in ng-skos is a JSON object with the following keys:

uri
URI of the terminology
prefLabel
object that maps language tags to preferred names of the terminology, each given as string.
altLabel
object that maps language tags to unordered arrays of alternative names, each given as string.
notation
ordered array of acronyms of similar short names, each given as string.
topConcepts
unordered array of strings with concept URIs and/or concept objects.
