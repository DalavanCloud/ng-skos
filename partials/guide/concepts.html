A concept in ng-skos is a JSON object with the following keys:

uri
URI of the concept, given as string
prefLabel
object that maps language tags to preferred labels, each given as string
altLabel
object that maps language tags to unordered arrays of alternative labels, each given as string
notation
partly ordered array of notations, each given as string. The first notation might be used as primary notation but multiple notations might also be used equivalently. The data type is assumed to be undefined or implicitly known. To support multiple notations of different type, additional concept keys might be introduced in a later version of ng-skos.
