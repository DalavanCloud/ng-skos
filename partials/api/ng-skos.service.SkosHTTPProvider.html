SkosHTTPProvider
service in module ng-skos

Description

Utility service to facilitate HTTP requests.

This service implements use of URL templates to perform HTTP requests with optional transformation of JSON responses and error handling. Directly use SkosConceptProvider and SkosConceptListProvider instead.

Configuration

  • url: URL template for requests
  • jsonp: enable JSONP
  • transform: custom transformation function to map expected response format

Methods

  • get([url]): perform a HTTP request
