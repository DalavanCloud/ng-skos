skosConceptList
directive in module ng-skos

Description

This directive displays a list of concepts with options to manipulate those lists.

Source code

The most recent source code of this directive is available at GitHub.

Usage

as attribute
<ANY skos-concept-list="{string}"
     on-select="{string}"
     template-url="{string}">
   ...
</ANY>

Parameters

ParamTypeDetails
skosConceptListstring

Object containing an array of concepts

onSelectstring

function handling the selection of one list item

templateUrlstring

URL of a template to display the concept list
