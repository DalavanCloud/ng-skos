skosConceptBrowser
directive in module ng-skos

Description

Scope

The following variables are added to the scope:

  • concept

Usage

as element:
<skos-concept-browser
       concept="{string}"
       typeahead="{string}"
       lookup-notation="{string}"
       search="{string}"
       template-url="{string}">
</skos-concept-browser>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
conceptstring

selected concept

typeaheadstring

OpenSearchSuggestions for typeahead

lookup-notationstring

SkosConceptProvider to look up by notation

searchstring

NOT SUPPORTED YET

template-urlstring

URL of a template to display the concept browser
