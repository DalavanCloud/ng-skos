skosBrowser
directive in module ng-skos

Description

Scope

The following variables are added to the scope, in addition to parameters:

  • notation
  • selectNotation

Limitations

By now, only getByNotation is supported.

Customization

The default template can be changed with parameter templateUrl.

Source code

The most recent source code of this directive is available at GitHub.

Usage

as element:
<skos-browser
       concept="{string}"
       suggest-concept="{string}"
       get-by-notation="{string}"
       template-url="{string}">
</skos-browser>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
conceptstring

selected concept

suggest-conceptstring

OpenSearchSuggestions for typeahead

get-by-notationstring

function to look up by notation (promise)

template-urlstring

URL of a template to display the concept browser
