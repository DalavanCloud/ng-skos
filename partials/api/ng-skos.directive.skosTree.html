skosTree
directive in module ng-skos

Description

Displays a hierarchical view of a concept and its transitive narrowers.

Customization

The default template can be changed with parameter templateUrl.

Usage

as attribute
<ANY skos-tree
     skos-tree="{string}"
     template-url="{string}">
   ...
</ANY>

Parameters

ParamTypeDetails
skos-treestring

Tree to display

template-urlstring

URL of a template to display the tree
