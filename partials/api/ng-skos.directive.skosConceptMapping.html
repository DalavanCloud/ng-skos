skosConceptMapping
directive in module ng-skos

Description

This directive displays two lists of concepts for the purpose of mapping from one concept scheme to another. In addition, it provides tools to customize and export those mappings.

Source code

The most recent source code of this directive is available at GitHub.

Usage

as attribute
<ANY skos-concept-mapping
     mapping="{string}"
     from="{string}"
     to="{string}"
     select="{string}"
     save-location="{string}"
     template-url="{string}">
   ...
</ANY>

Parameters

ParamTypeDetails
mappingstring

Mapping to display

fromstring

Name of the current source scheme

tostring

Name of the current target scheme

selectstring

function to export a single concept scheme

saveLocationstring

function to save the current mapping

templateUrlstring

URL of a template to display the mapping
