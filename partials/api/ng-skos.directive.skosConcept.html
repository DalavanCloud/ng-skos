skosConcept
directive in module ng-skos

Description

Display a concept. Changes on the concept object are reflected by changes in the scope variables so the display is updated automatically.

Scope

The following variables are added to the scope:

  • ancestors (array of concepts)
  • prefLabel (object of strings)
  • altLabel (object of array of strings)
  • notation (string)
  • note (object of array of strings)
  • broader (array of concepts)
  • narrower (array of concepts)
  • related (array of concepts)

In addition the helper method isEmptyObject is provided to check whether an object is empty.

Customization

The default template can be changed with parameter templateUrl.

Usage

as attribute
<ANY skos-concept
     skos-concept="{string}"
     language="{string}"
     skos-click="{string}"
     template-url="{string}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails

function to call when a connected concept is clicked

template-urlstring
