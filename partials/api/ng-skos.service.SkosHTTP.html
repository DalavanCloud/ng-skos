SkosHTTP
service in module ng-skos

Description

Utility service to facilitate HTTP requests.

This service implements use of URL templates to perform HTTP requests with optional transformation of JSON responses and error handling.

The service is not related to JSKOS but used as utility in ng-skos. A future release of ng-skos may drop it favor of $http or use $resource.

Configuration

  • url: URL requests
  • jsonp: enable JSONP (true/false/0/1/name)
  • transform: custom transformation function to map expected response format

Methods

  • get([url]): perform a HTTP request
