jsonText
directive in module json-text

Description

This directive can be used to display or edit an object in JSON syntax:

<textarea json-text ng-model="myObject"/>

Usage

as element:
<json-text
       ng-model="{string}"
       json-valid="{string}">
</json-text>
as attribute
<ANY json-text
     ng-model="{string}"
     json-valid="{string}">
   ...
</ANY>

Parameters

ParamTypeDetails
ngModelstring

angular expression to bind to.

json-validstring

AngularJS expression to bind parsing status to.
