skosNotes
directive in module ng-skos

Description

Shows the documentary notes of a concept. A preferred language can be selected with parameter lang. If no language is selected, an arbitrary language with given notes is chosen.

Scope

The variables scopeNote, definition, example, and historyNote are added to the scope, if given es keys of the skos-notes parameter object.

Customization

The default template can be changed with parameter templateUrl.

Usage

as attribute
<ANY skos-notes
     skos-notes="{string}"
     [lang="{string}"]
     template-url="{string}">
   ...
</ANY>

Parameters

ParamTypeDetails
skos-notesstring

Expression with multilingual notes data

lang
(optional)
string

preferred language to show notes in.

template-urlstring

URL of a template to display the concept
