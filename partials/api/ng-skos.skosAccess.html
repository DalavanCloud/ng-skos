skosAccess
service in module ng-skos

Description

Look up concepts and terminologies by URI.

An skosAccess service should be used as layer to access concepts and terminologies. The service adds caching and additional methods to expand a list of URIs to a list of concept or terminology objects.

See CocodaClient for a sample source.

var ddc = CocodaTerminology("http://example.org/ddc/");
var ddcAccess = new skosAccess(ddc);
