skosLabel
directive in module ng-skos

Description

Displays the preferred label of a concept. Changes on the preferred label are reflected in the display.

Usage

as attribute
<ANY skos-label
     skos-label="{string}"
     [lang="{string}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
skos-labelstring

Assignable angular expression with concept data to bind to.

lang
(optional)
string

optional language. If not specified, an arbitrary preferred labels is selected.
