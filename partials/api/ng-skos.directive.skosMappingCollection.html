skosMappingCollection
directive in module ng-skos

Description

This directive displays mapping tables between concepts of two concept schemes.

Source code

The most recent source code of this directive is available at GitHub.

Usage

as attribute
<ANY skos-mapping-collection
     skos-mapping-collection="{string}"
     use-mapping="{string}"
     template-url="{string}">
   ...
</ANY>

Parameters

ParamTypeDetails
skos-mapping-collectionstring

Mapping to display

use-mappingstring

function to handle mappings selected from within this template

template-urlstring

URL of a template to display the mapping
