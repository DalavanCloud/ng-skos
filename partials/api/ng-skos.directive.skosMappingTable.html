skosMappingTable
directive in module ng-skos

Description

This directive displays mappings between concepts of two concept schemes in a table format.

Source code

The most recent source code of this directive is available at GitHub.

Usage

as attribute
<ANY skos-mapping-table
     skos-mapping-table="{string}"
     select-mapping="{string}"
     template-url="{string}">
   ...
</ANY>

Parameters

ParamTypeDetails
skos-mapping-tablestring

Mapping to display

select-mappingstring

function to handle mappings selected from within this template

template-urlstring

URL of a template to display the mapping
