skosList
directive in module ng-skos

Description

This directive displays a list of concepts with options to manipulate those lists.

Customization

The default template can be changed with parameter templateUrl.

Usage

as attribute
<ANY skos-list
     concepts="{string}"
     on-select="{string}"
     can-remove="{string}"
     show-labels="{string}"
     template-url="{string}">
   ...
</ANY>

Parameters

ParamTypeDetails
conceptsstring

array of JSKOS concepts to display

onSelectstring

function handling the selection of one concept

canRemovestring

support a removeConcept method to remove concepts

showLabelsstring

chose, if concept labels should be shown as well as notations

templateUrlstring

URL of a template to display the concept list
