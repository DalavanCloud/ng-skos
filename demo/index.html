skos-label

Used as attribute to emit the preferred label of a concept. The lang attribute of the directive element is read and set accordingly. If no label of the required language exists, another language is selected.

skos-mapping-collection

skosMappingCollection is a directive for displaying a collection of existing mappings between the notations of different concept schemes. In this example, the used template sorts the mappings into lists and passes those on to skosMappingTable, which displays each table structure.
The directive also supports an additional parameter "use-mapping", with which a function controlling the selection of a mapping in the "Actions" column can be handled.
